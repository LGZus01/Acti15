HTML5

Brevemente, el HTML 5 representa las siglas en inglés de HyperText Markup Language versión 5 (lenguaje de marcas de hipertexto versión 5), siendo la quinta revisión del lenguaje básico de la World Wide Web, HTML. Se publicó en 2014 y presenta entre sus particularidades, dos variantes de sintaxis, una clásica (text/html) conocida como HTML5 y una variante XHTML conocida como sintaxis XHTML5 que deberá servirse con sintaxis XML (application/xhtml+xml). Así pues, HTML 5 permite incorporar a las páginas ficheros RDF / OWL (con meta información) para describir relaciones entre los términos utilizados y ofrece versatilidad en el manejo y animación de objetos simples, imágenes, entre otros. Entre algunas características del lenguaje HTML 5 podemos mencionar las siguientes: Añade nuevas etiquetas para manejar la Web semántica (Web 3.0). Presenta una cantidad de nuevos elementos y algunos atributos que reflejan el uso típico de los sitios web más modernos. Introduce nuevas formas de escribir código, así que la sintaxis varía notablemente. La extensión del archivo en el lenguaje HTML 5 puede ser .html, .htm. .xhtml, .xht o .xml. De acuerdo a las especificaciones presentadas, las diferencias entre HTML y HTML 5 son: HTML es el lenguaje predominante para la elaboración de páginas web mientras que HTML 5 todavía está en proceso de ser adoptado por muchos desarrolladores. HTML se escribe en forma de etiquetas rodeadas por corchetes angulares (‹›) y puede describir hasta un cierto punto, la apariencia de un documento mientras que HTML 5 añade nuevas etiquetas para manejar la Web semántica (Web 3.0).
