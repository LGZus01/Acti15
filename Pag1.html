HTML

Fundamentalmente, HTML son las siglas en inglés de HyperText Markup Language (lenguaje de marcas de hipertexto) y es un lenguaje utilizado para la elaboración de páginas web lanzado oficialmente en el año 1993. Asi pues, el lenguaje HTML se considera el lenguaje web más importante, siendo su invención crucial en la aparición, desarrollo y expansión de la World Wide Web (WWW). Por otra parte, este es el estándar que se ha utilizado más en la visualización de las páginas web y actualmente es el que todos los navegadores han adoptado. Entre algunas características del lenguaje HTML podemos mencionar las siguientes: Se escribe en forma de etiquetas rodeadas por corchetes angulares (‹›) y puede describir hasta un cierto punto, la apariencia de un documento. Cada atributo y contenido tiene ciertas restricciones para que se considere válida al documento HTML y todas sus variantes abren las instrucciones con ‹› y cierran con ‹›, por ejemplo ‹p›text‹/p›. La extensión del archivo en el lenguaje HTML es .html o .htm. El lenguaje HTML está diseñado básicamente para desplegar datos y está enfocado a cómo se ven esos datos.
